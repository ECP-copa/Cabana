Detailed Description

template<class ViewType>
struct Cajita::VectorDivergenceG2P< ViewType >

Grid-to-point vector value functor.

Interpolates the divergence of a vector function from entities to points with a given multiplier such that:

f_p = multiplier * \sum_d \sum_{ijk} grad_weight_{pijkd} * f_{ijkd}
