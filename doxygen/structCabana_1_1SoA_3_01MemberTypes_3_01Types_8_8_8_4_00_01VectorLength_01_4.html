Detailed Description

template<typename... Types, int VectorLength>
struct Cabana::SoA< MemberTypes< Types...>, VectorLength >

Struct-of-Arrays.

A struct-of-arrays (SoA) is composed of groups of statically sized arrays. The array element types, which will be composed as members of the struct, are indicated through the Types parameter pack. If the types of the members are contiguous then the struct itself will be contiguous. The vector length indicates the static length of each array.
