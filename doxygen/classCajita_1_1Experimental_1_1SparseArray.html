Detailed Description

template<class DataTypes, class DeviceType, class EntityType, class MeshType, class SparseMapType>
class Cajita::Experimental::SparseArray< DataTypes, DeviceType, EntityType, MeshType, SparseMapType >

Sparse array of field data on the local sparse mesh; Array data is stored in AoSoA manner, with each tile being the SoA element.
