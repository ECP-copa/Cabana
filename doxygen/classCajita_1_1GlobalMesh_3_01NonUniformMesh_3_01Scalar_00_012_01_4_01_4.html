Detailed Description

template<class Scalar>
class Cajita::GlobalMesh< NonUniformMesh< Scalar, 2 > >

Global mesh partial specialization for 2D non-uniform mesh.

Template Parameters
ScalarMesh floating point type.

Non-uniform meshes have a list of node locations for each spatial dimension which describe a rectilinear mesh that has arbitrary cell sizes - each cell can possibly be different.
