Detailed Description

template<class ViewType>
struct Cabana::Grid::ScalarValueG2P< ViewType >

Grid-to-point scalar value functor.

Interpolates a scalar function from entities to points with a given multiplier such that:

f_p = multiplier * \sum_{ijk} weight_{pijk} * f_{ijk}
